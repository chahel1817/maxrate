# ⚡ MaxRate Frontend
### High-Performance Dashboard & UI

This is the Next.js 14 frontend for the MaxRate API Rate Limiting platform.

## 🌈 Aesthetics

*   **Glassmorphism-inspired UI**: Premium cards and micro-animations for sub-second visual feedback.
*   **Dynamic Data Visuals**: Real-time traffic overlook graphs with automated polling.
*   **Accessible Components**: Lucide iconography and semantic HTML.

## 🛠️ Tech Stack

*   **Framework**: [Next.js](https://nextjs.org/) (App Router, Turbopack).
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/) with a curated dark-mode palette.
*   **Animations**: [Framer Motion](https://www.framer.com/motion/).
*   **Icons**: [Lucide React](https://lucide.dev/).
*   **Fetch Wrapper**: Custom axios-like wrapper with error interception.

## 📦 Getting Started

1. Navigate to this directory: `cd frontend`.
2. Install: `npm install`.
3. Dev server: `npm run dev`.

The frontend communicates with the MaxRate backend at `http://localhost:8080`.
Ensure those services are running for a complete data-driven experience.
